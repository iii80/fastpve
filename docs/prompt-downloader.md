这个项目是给 PVE 系统快速安装虚拟机。
比如安装 Window10 Windows11 Ubuntu iStoreOS 等等。
安装系统都需要给用户一个命令行的操作界面，而这个已经在 cmd/fast 里面得到实现。
一键安装系统还有个核心是，能下载到对应的操作系统。而这个目前一部分依赖 quickget 实现，
一部分直接写到了 cmd/fast 里面。
这个下载的逻辑，最好能从 cmd/fast 里面分开，到一个独立的模块，比如 vm-downer 中。

如果觉得 vm-downer 命名不好，也可以改个名字。

请优化当前的代码实现，把多个虚拟机的下载独立到 vm-downer 中，方便对下载进行单元测试。
